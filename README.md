# Ruboty::Vimhelp

Show vim help.

## Usage

```ruby
# Gemfile
gem 'ruboty-vimhelp'
```

```
# Say ":help <keyword>"
> ruboty :help i
<insert>    or                *i* *insert* *<Insert>*
i            カーソルの前にテキストを[count]回挿入する。
            挿入モード中のCTRL-O |i_CTRL-O| で使うときはカウントは
            サポートされていない。
e
```
