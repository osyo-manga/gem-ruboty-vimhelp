# Ruboty::Vimhelp

Show vim help.

## Usage

```ruby
# Gemfile
gem 'ruboty-through_ruboty_name'
```

```
# Say ":help <keyword>"
:help i
<insert>    or                *i* *insert* *<Insert>*
i            カーソルの前にテキストを[count]回挿入する。
            挿入モード中のCTRL-O |i_CTRL-O| で使うときはカウントは
            サポートされていない。
e
```
